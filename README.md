# AxioLocusZoom
R ShinyApp HTMLWidget using the locuszoom.js library 

1. Install devtools
```R
install.packages("devtools")
```
2. Load the devtools package
```R
library(devtools)
```
3. Install using `install_github()` command
```R
install_github("jjsayleraxio/AxioLocusZoom")
```
